Documentation

The DAM2TWIN project refers to the final undergraduate capstone work by research students at Institut Teknologi Nasional Malang. The purpose of the project was to design and develop a web-based Building Information Modeling (BIM) system integrated with 3D GIS for irrigation asset management at Bendung Pataruman. This documentation summarizes the objectives, methodology, results, and impacts as outlined in the thesis.

ðŸ”Ž Abstract

This project aims to realize digital transformation in managing irrigation assets using integrated geospatial technologies. The development combines photogrammetric survey data, 3D modeling, web GIS frameworks, and interactive documentation tools. The result is an accessible platform supporting monitoring, visualization, and sustainable asset management of Bendung Pataruman.
